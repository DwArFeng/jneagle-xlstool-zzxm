## 项目简介

处理机加工进度表，导出在制零件加工完成情况表。

* 使用GUI与用户对接，界面简洁美观，方便操作。
* 支持带密码的xls数据表以及xlsx数据表。
* 大部分的导入与导出操作细节由配置决定，可以进行柔性化调整。
* 带有日志以及国际化功能。
